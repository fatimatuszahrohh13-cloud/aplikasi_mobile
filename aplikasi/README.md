# AgriGrow

AgriGrow adalah aplikasi seluler modern dan ramah pengguna yang dibangun dengan Flutter, dirancang untuk membantu pengguna dalam mengelola dan melacak aktivitas pertanian mereka.

## Fitur

- **Beranda (Dashboard)**: Dapatkan ringkasan cepat tentang aktivitas pertanian Anda.
- **Pemindaian Tanaman**: Gunakan fitur pemindaian untuk mengidentifikasi tanaman, mendeteksi penyakit, atau menganalisis kesehatan tanaman.
- **Perpustakaan**: Akses perpustakaan komprehensif yang berisi berbagai sumber daya, informasi tanaman, dan panduan pertanian.
- **Kalender**: Kelola jadwal penanaman, penyiraman, dan pemanenan Anda secara efisien.

## Memulai

Proyek ini merupakan titik awal untuk aplikasi Flutter.

### Prasyarat

- Flutter SDK (versi `^3.11.0` atau lebih tinggi)
- Dart SDK
- Android Studio / VS Code dengan ekstensi Flutter

### Instalasi

1. Clone repositori:
   ```bash
   git clone <url-repositori>
   ```
2. Arahkan ke direktori proyek:
   ```bash
   cd fatim
   ```
3. Instal dependensi:
   ```bash
   flutter pub get
   ```
4. Jalankan aplikasi:
   ```bash
   flutter run
   ```

## UI & Tema

AgriGrow menampilkan tema bernuansa alam yang memanfaatkan prinsip-prinsip desain Material 3.

- **Warna Hijau Utama**: `#2E7D32`
- **Warna Permukaan (Surface)**: `#F4F7F4`

## Struktur Proyek

- `lib/main.dart` - Titik masuk (entry point) dari aplikasi yang mengatur tema dan perutean (routing).
- `lib/screens/` - Berisi layar antarmuka utama (misalnya: Beranda, Pindai, Perpustakaan, Kalender).
